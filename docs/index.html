Ghostpad is an interface that allows you to load and interact with text generation AI models on your PC.
Mobile First Customizable UI
Ghostpad was designed from the ground up to be one of the easiest ways to interact with an AI from your touch devices. It supports custom themes and in-app settings for local fonts or Google Fonts.
Based on KoboldAI
KoboldAI is the parent project of Ghostpad, from which it inherits most of its powerful features. Ghostpad is a React-based alternative which seeks to move quickly on experimental features while contributing mutually beneficial changes upstream.
Two Way Speech
Turn on audio input to speak to your AI, or turn on audio output to enable text-to-speech features with customizable voices. If you enable both, you can enjoy a hands-free conversation.
AutoAWQ Support
It is a goal of this project to quickly implement support for the latest model formats, accepting the potential stability tradeoffs. Ghostpad can be used to load quantized models in the GPTQ, ExLlama, ExllamaV2, or AWQ formats.
